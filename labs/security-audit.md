# Conduct a Security Audit

## Overview

This project focuses on performing a security controls and compliance assessment for Botium Toys, a small business with an expanding online presence and increasing security needs.

The objective was to evaluate existing security controls, identify missing safeguards, assess compliance gaps, and provide practical recommendations to improve the organization's overall security posture.

---

## Scope of Assessment

The audit reviewed the following areas:

- Administrative controls
- Technical controls
- Physical controls
- Compliance with PCI DSS
- GDPR requirements
- SOC 1 / SOC 2 best practices

The goal was to determine whether appropriate controls were already in place and identify areas requiring immediate improvement.

## Recommendations

Based on the assessment, Botium Toys should prioritize the following improvements:

### Immediate Priorities

- Implement Least Privilege access controls
- Deploy an Intrusion Detection System (IDS)
- Enable encryption for sensitive business and customer data
- Establish secure password management systems
- Improve secure backup procedures
- Strengthen monitoring of legacy systems

### Compliance Improvements

- Improve PCI DSS compliance by securing payment card data
- Strengthen GDPR compliance for customer privacy and breach notification
- Establish stronger SOC access control policies
- Improve data classification and inventory management

These improvements would significantly reduce organizational risk, improve regulatory compliance, and strengthen the company’s overall security posture.
